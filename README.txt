The Scheduling Application is a JavaFX application providing abilities for users
to create/manage appointments of customers in different timezones. It also enables users to create/modify customers and generate different reports based on certain conditions.

Application Version: 1.14

IDE: IntelliJ Community 2021.1.3 x64
JDK: Java SE 17.0.1
JavaFX: JavaFX-SDK-17.0.1
MySQL: mysql-connector-java-8.0.25

Instructions:
1. Install JDK 17 or higher.
2. Install & configure JavaFX.
3. Import project to IntelliJ IDE.
4. Build & Run project from IntelliJ: Run -> Run "Main".
5. Log into application: Username:test Password:test

Note:
1. The javadoc files are located at "javadoc".
2. Language translation in English & French for the Login window only.
3. 15-minute appointment reminder feature is specific to the user logged in.
4. Lambda example #1 can be found at src.Controller.CustsController.initialize().
5. Lambda example #2 can be found at src.Controller.MoTypReportController.genReport().
6. The additional report for section A.3.f is the location report to lists appointments sorted in locations and start times.
